# Description:

FlightPing is my first attempt in building an iPhone application that lets you check you flight status. At the moment it only supports Cebu Pacific airlines.


# Requirements:

- Pyramid
- PhantomJS
- Xcode 4.x

# Running:

- Run the middleware: python middleware.py
- Run xcode project
